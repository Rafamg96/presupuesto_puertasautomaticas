## Flujo de Usuario

### Paso 1: Llenar Formulario
```
Usuario → Completa datos → Selecciona opciones → Añade accesorios
```

### Paso 2: Generar Presupuesto
```
Click en "Generar Presupuesto" → Cálculo automático → Vista previa
```

### Paso 3: Exportar o Imprimir
```
Vista previa → Click "Descargar PDF" → PDF generado → Guardado local
         o
Vista previa → Click "Imprimir" → Diálogo de impresión del navegador
```

## Interacciones

### Selección de Tipo de Puerta
```
Estado Inicial:   [  Corredera  ]
Hover:           [  Corredera  ]  (borde gris claro)
Seleccionada:    [━━Corredera━━]  (borde azul + fondo azul claro)
```

### Toggle de Urgencia
```
Desactivado:  ☐ Instalación urgente
Activado:     ☑ Instalación urgente  (fondo destacado)
```

### Botón Principal
```
Normal:     [  🧮 Generar Presupuesto  ]
Hover:      [  🧮 Generar Presupuesto  ]  (sombra + color más oscuro)
Click:      [  🧮 Generar Presupuesto  ]  (animación de escala)
```

## Estados de la Aplicación

### Estado 1: Formulario Vacío
```
Formulario → Todos los campos en blanco → Botón "Generar" activo
```

### Estado 2: Formulario Completo
```
Formulario → Campos llenos → Validación OK → Listo para generar
```

### Estado 3: Vista Previa
```
Documento → Presupuesto mostrado → Botones "Volver", "Imprimir", "PDF"
```

### Estado 4: Descargando PDF
```
Vista Previa → Click "Descargar PDF" → Generación → Descarga automática
```
